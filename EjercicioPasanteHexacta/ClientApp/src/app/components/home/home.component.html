Buenas!

Esta es una aplicación integradora que utiliza:

  • Frontend: Angular + HTML
  • Backend: .NET Core
  • Base de datos: SQL Server (Entity Framework Core)

Para ver las a las personas o agregar una nueva haga click en Lista de Personas o Agregar Persona respectivamente.

Es probable que la URL del servidor generada difiera de la utilizada durante el desarrollo de este proyecto (https://localhost:7055), por lo que sería conveniente modificar la propiedad baseUrl ubicada en ClientApp > src > environments > environment.ts por la URL del servidor actual.
